Experience

In my experience with Computer Science, I have taken classes such as Intro to Program Design and Concepts, Discrete Structures of Computing (Discrete Math), Data Structures and Algorithms, Computer Organization, and Programming Languages, and am currently taking Intro to Computer Systems and Programming Studio. These classes have taught me the basics of C++, Scheme, Assembly, and Java, as well as given me background in program and algorithm design and analysis, processor design, and the basics of how programming languages operate and communicate with the processor. Along with the languages previously mentioned, I have some experience working with C# and Python, although not as extensive as my experience with the others.

Some of the projects I have worked on include:

  • Implementation of abstract data types such as stacks, queues, and linked lists
  • Binary search trees
  • Sorting algorithms
  • Assembler that parses instructions and converts them to binary
  • Virtual CPU using assembly
